## นัดหมาย / ส่งเรื่องให้ทนาย

### ขั้นตอน

1. **รวบรวมข้อมูล** — ถ้ายังไม่ครบ ให้ถามผู้ใช้:

| ข้อมูล | จำเป็น | ตัวอย่าง |
|--------|--------|---------|
| ชื่อ-นามสกุล | ✅ | สมชาย ใจดี |
| เบอร์โทร | ✅ | 081-xxx-xxxx |
| อีเมล | แนะนำ | example@mail.com |
| ประเภทคดี | ✅ | labor / contract / criminal / other |
| รายละเอียดเบื้องต้น | ✅ | ถูกเลิกจ้างไม่เป็นธรรม ทำงานมา 5 ปี |

2. **ยืนยันข้อมูล** — แสดงข้อมูลให้ผู้ใช้ตรวจสอบก่อนส่ง:

```
📋 ข้อมูลที่จะส่ง:
- ชื่อ: [ชื่อ]
- โทร: [เบอร์]
- ประเภท: [ประเภทคดี]
- รายละเอียด: [สรุป]

ยืนยันส่งเรื่อง? (ใช่/ไม่)
```

3. **สร้าง Lead** — เรียก MCP tool `case_intake` ด้วยข้อมูลที่รวบรวม

4. **แจ้งผลและขั้นตอนถัดไป**:

```
## ✅ ส่งเรื่องเรียบร้อย

📌 หมายเลขเรื่อง: [LEAD-xxx]
📅 ทีมทนายจะติดต่อกลับภายใน 24 ชั่วโมง

### 📎 เอกสารที่ควรเตรียม
- [รายการเอกสารตามประเภทคดี]
```

### เอกสารตามประเภทคดี

**แรงงาน (labor):**
- สัญญาจ้างงาน
- สลิปเงินเดือน 3-6 เดือนล่าสุด
- หนังสือเลิกจ้าง / หนังสือเตือน
- หลักฐานการจ่าย/ไม่จ่ายค่าชดเชย

**สัญญา (contract):**
- สัญญาที่เกี่ยวข้อง
- หลักฐานการชำระเงิน
- หลักฐานความเสียหาย
- หนังสือโต้ตอบ

**อาญา (criminal):**
- บันทึกประจำวัน / ใบแจ้งความ
- หลักฐานเกี่ยวกับเหตุการณ์
- ข้อมูลพยาน

### กฎ
- ห้ามส่งเรื่องก่อนได้รับยืนยันจากผู้ใช้
- ข้อมูลส่วนบุคคลเป็นความลับ
- ถ้าเป็นเรื่องเร่งด่วน (ถูกจับ, มีหมายศาล) แนะนำโทรสายด่วนทันที
