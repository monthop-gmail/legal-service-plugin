## เปรียบเทียบทางเลือกทางกฎหมาย

### ขั้นตอน

1. **วิเคราะห์สถานการณ์** จาก: $ARGUMENTS
2. **ค้นหากฎหมาย** — เรียก `legal_search` ที่เกี่ยวข้องกับแต่ละทางเลือก
3. **ประเมินค่าใช้จ่าย** — เรียก `fee_estimate` สำหรับแต่ละทางเลือก
4. **เปรียบเทียบ**

### รูปแบบการตอบ

```
## ⚖️ เปรียบเทียบทางเลือก

### สถานการณ์
[สรุปสั้นๆ]

| เกณฑ์ | ทางเลือก A | ทางเลือก B | ทางเลือก C |
|-------|-----------|-----------|-----------|
| 📝 วิธีการ | เจรจา | ร้องเรียน กคพ. | ฟ้องศาลแรงงาน |
| 💰 ค่าใช้จ่าย | X บาท | ฟรี | X บาท |
| ⏱️ ระยะเวลา | 1-2 สัปดาห์ | 1-3 เดือน | 3-12 เดือน |
| 📊 โอกาสสำเร็จ | สูง/กลาง/ต่ำ | ... | ... |
| 💵 ผลลัพธ์ที่คาดหวัง | X บาท | X บาท | X บาท |
| ✅ ข้อดี | ... | ... | ... |
| ❌ ข้อเสีย | ... | ... | ... |

### 💡 คำแนะนำ
[แนะนำทางเลือกที่เหมาะสมที่สุดตามสถานการณ์ พร้อมเหตุผล]

### 🗺️ แผนการดำเนินงาน (ทางเลือกที่แนะนำ)
1. [ขั้นตอนที่ 1]
2. [ขั้นตอนที่ 2]
3. ...
```

### ทางเลือกตามประเภทคดี

**แรงงาน:**
- เจรจาตรง / ไกล่เกลี่ย
- ร้องเรียนพนักงานตรวจแรงงาน
- ร้องเรียน กคพ. (คณะกรรมการแรงงานสัมพันธ์)
- ฟ้องศาลแรงงาน (ไม่ต้องเสียค่าฤชาธรรมเนียม)

**สัญญา:**
- เจรจา / ไกล่เกลี่ย
- อนุญาโตตุลาการ
- ฟ้องศาลแพ่ง
- ฟ้องศาลแผนกคดีผู้บริโภค (ถ้าเข้าเงื่อนไข)

### กฎ
- ให้ข้อมูลอย่างเป็นกลาง ไม่ลำเอียง
- ระบุ "โอกาสสำเร็จ" ว่าเป็นการประเมินเบื้องต้นเท่านั้น
- แนะนำปรึกษาทนายก่อนตัดสินใจเสมอ
