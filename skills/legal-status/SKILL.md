## ดูสถานะเรื่อง

### ขั้นตอน

1. เรียก MCP tool `list_leads` เพื่อดึงข้อมูล Lead ทั้งหมด
2. ถ้าผู้ใช้ระบุหมายเลข LEAD → แสดงรายละเอียดเฉพาะเรื่อง
3. ถ้าไม่ระบุ → แสดงตารางสรุปทั้งหมด

### รูปแบบการตอบ

**กรณีดูทั้งหมด:**
```
## 📊 สรุป Lead ทั้งหมด

| # | หมายเลข | ชื่อ | ประเภท | สถานะ | วันที่สร้าง |
|---|---------|------|--------|-------|-----------|
| 1 | LEAD-xxx | ... | แรงงาน | ใหม่ | ... |

รวม: X เรื่อง | ใหม่: X | กำลังดำเนินการ: X | เสร็จสิ้น: X
```

**กรณีดูเฉพาะเรื่อง:**
```
## 📌 รายละเอียด [LEAD-xxx]

- ชื่อ: ...
- ประเภท: ...
- สถานะ: ...
- วันที่สร้าง: ...
- รายละเอียด: ...
```

### สถานะ
| สถานะ | ความหมาย |
|-------|---------|
| 🆕 new | เรื่องใหม่ รอทีมทนายติดต่อ |
| 📞 contacted | ติดต่อลูกค้าแล้ว |
| ✅ qualified | ประเมินแล้ว พร้อมรับเรื่อง |
| 🤝 converted | รับเป็นลูกความแล้ว |
