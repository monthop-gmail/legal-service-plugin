## ร่างหนังสือทางกฎหมาย

### ประเภทหนังสือที่รองรับ

| ประเภท | คำสั่ง | ใช้เมื่อ |
|--------|--------|---------|
| หนังสือทวงถาม | `/legal-draft ทวงถาม [รายละเอียด]` | ทวงค่าชดเชย / เงินค้าง |
| หนังสือร้องเรียน | `/legal-draft ร้องเรียน [รายละเอียด]` | ร้องเรียนหน่วยงานรัฐ |
| บันทึกข้อตกลง | `/legal-draft ข้อตกลง [รายละเอียด]` | ตกลงเจรจายอมความ |
| หนังสือตอบโต้ | `/legal-draft ตอบโต้ [รายละเอียด]` | ตอบหนังสือทวงถาม |

### ขั้นตอน

1. **ระบุประเภท** จาก: $ARGUMENTS
2. **รวบรวมข้อมูล** ที่จำเป็น:
   - ผู้ส่ง (ชื่อ, ที่อยู่)
   - ผู้รับ (ชื่อบริษัท/บุคคล, ที่อยู่)
   - ข้อเท็จจริง (วันที่เริ่มงาน, วันที่เลิกจ้าง, จำนวนเงิน ฯลฯ)
3. **ค้นหากฎหมาย** — เรียก `legal_search` เพื่ออ้างอิงมาตรา
4. **ร่างหนังสือ** ตามรูปแบบด้านล่าง
5. **บันทึกไฟล์** เป็น `.md` หรือ `.txt` ใน working directory

### รูปแบบหนังสือทวงถาม

```
                                        [สถานที่]
                                        [วันที่]

เรื่อง  [หัวเรื่อง]
เรียน  [ผู้รับ]

        ตามที่ข้าพเจ้า [ชื่อ] ได้ [ข้อเท็จจริง]...

        ข้าพเจ้าขอแจ้งให้ทราบว่า ตามพระราชบัญญัติ [ชื่อ พ.ร.บ.] มาตรา [X]
กำหนดว่า [สาระสำคัญ]...

        ข้าพเจ้าจึงขอให้ท่าน [สิ่งที่ต้องการ] ภายในกำหนด [X] วัน
นับแต่วันที่ได้รับหนังสือฉบับนี้ หากพ้นกำหนดแล้วยังไม่ได้รับการติดต่อ
ข้าพเจ้ามีความจำเป็นต้องดำเนินการตามกฎหมายต่อไป

        จึงเรียนมาเพื่อทราบและดำเนินการ

                                        ขอแสดงความนับถือ

                                        [ลงชื่อ]
                                        ([ชื่อ-นามสกุล])
                                        โทร. [เบอร์โทร]
```

### กฎ
- ใช้ภาษาราชการ สุภาพ ชัดเจน
- อ้างอิงมาตรากฎหมายที่ถูกต้องเสมอ
- ระบุ disclaimer ว่า "ควรให้ทนายตรวจสอบก่อนส่งจริง"
- ห้ามใส่ข้อมูลเท็จหรือข่มขู่เกินกว่าสิทธิตามกฎหมาย
- กำหนดระยะเวลาที่สมเหตุสมผล (ปกติ 7-30 วัน)
